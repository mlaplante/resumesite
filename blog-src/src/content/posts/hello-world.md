This is the inaugural post on my blog. I built this using [Astro](https://astro.build), a modern static site generator that outputs pure HTML with zero JavaScript by default.

## Why a Blog?

I wanted a space to share:

- **Project updates** — what I'm building and why
- **Thought leadership** — observations from 15+ years in security and engineering
- **Dev sessions** — interesting problems solved with AI-assisted development

## The Stack

The blog is powered by Markdown files compiled to static HTML by Astro. It lives alongside my existing resume site with shared styling — same fonts, same colors, same nav.
